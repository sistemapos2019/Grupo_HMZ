Sistema de Ventas Jupiter POS

Gestione su Negocio Fácilmente


Jupiter POS es un sistema para efectuar ordenes de los clientes en restaurantes o negocios dedicados para venta de alimentos preparados. Capaz de administrar las funciones escenciales que un negocio realiza como el manejo de mesas, meseros, inventarios, etc. Además mostar estadisticas de lo vendido y creación de libros contables.


Características

Utilidad

Jupiter POS ofrece una manera funcional y sencilla de manejar cualquier negocio dedicado a la venta de alimentos preparados o restaurantes. Facilitando la gestión de mesas, meseros, platillos y resumenes contables.

Diseño

Jupiter POS ofrece un diseño totalmente responsive y amigable con el
usuario además de contar con una paleta de colores agradables.

Portabilidad


Por su diseño web es facilmente acesible desde cualquier dispositivo ya sea escritorio o movil.

INSTALACIÓN

Jupiter POS es licencia MIT, disponible para el uso de cualquier persona.

Para la instalación de Jupiter POS es necesario contar con el software XAMPP. Puede descargarlo dando clic aquí

Descarga el Sistema Jupiter POS dando clic aquí. Y da clic en el boton que dice Clone or Download y descarga el sistema como Zip.

Luego de descomprimir el sistema, se coloca dentro de la carpeta htcdocs en la ruta de instalación de xampp en el disco duro.

Busca el archivo pos.sql  dentro de los archivos del proyecto y lo instala en el apartado de MySQL de Xampp

Despues de instalar la base de datos, da clic en start a MySQL y Apache.

Por último puede hacer uso del software entrando a la dirección:
 http://localhost/dsi-backend/Dashboard
